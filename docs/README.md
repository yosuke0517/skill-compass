# Skill Compass ドキュメント

このディレクトリには、Skill Compassの公開可能な仕様、設計、実装計画、進捗記録を保存する。

## 言語ポリシー

- ToC向けUIの標準言語は英語とする。
- ToC向けUIは将来の多言語対応を前提とし、日本語へ切り替えられるようにする。
- admin向けUIの標準言語は日本語とする。
- README、仕様書、設計書、実装計画、進捗記録は日本語で記述する。
- コード、識別子、API名、OAuth scope、CLI commandなど、正確性が重要な技術表現は英語のまま使用できる。
- 既存の英語implementation planは履歴資料として原文を維持する。新規文書と、継続的に更新する文書には日本語ポリシーを適用する。

## 公開範囲

公開ドキュメントには、認証情報、API key、個人アカウント、非公開Source一覧、ローカル環境固有の情報、非公開の運用手順を含めない。
