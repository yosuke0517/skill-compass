# Skill Compass

Skill Compassは、AI支援時代のエンジニアに向けた個人用の成長支援アプリです。

適応型クイズ、スキル追跡、出典に基づく学習、Markdownによる知識exportを組み合わせ、エンジニアとしての成長を可視化し、継続的に振り返れるようにします。
